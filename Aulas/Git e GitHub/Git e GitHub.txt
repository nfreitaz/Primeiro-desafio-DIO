# Git

​	Git é um sistema de controle de versão gratuito. Foi desenvolvido por Linus Torvals e é utilizado para acompanhar o desenvolvimento e alterações em código-fonte dos projetos, também é possível verificar os usuários que fizeram essas alterações e quais foram feitas.



## GitHub

​	O Github é um serviço online onde você pode hospedar os repositórios do Git, ou seja, um repositório remoto.

​	Possuindo várias ferramentas que permite a integração dos projetos e o seu compartilhamento entre equipes.
